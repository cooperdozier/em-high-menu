Menu with Full Set of PureCSS Framework

Everything works essentially the same with the whole set of the PureCSS Framework, as seen on this page. The only pieces that are noticeably different are the capitalization of the left-top site title link and the code in the <pre> tag below, had I not restyled the pre tag. But the pre tag has no effect on the menu itself. You can find more on Pure at https://purecss.io. To use it, the simplest way is to add within HTML <head> element the following (unless you plan to be working off-line): 

<link rel="stylesheet" href="https://unpkg.com/purecss@2.0.5/build/pure-min.css" integrity="sha384-LTIDeidl25h2dPxrB2Ekgc9c7sEC3CWGM6HeFmuDNUjX76Ert4Z4IY714dhZHPLd" crossorigin="anonymous">
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="https://unpkg.com/purecss@2.0.5/build/grids-responsive-min.css">

That should be added above your custom stylesheet <link> tags. Also, to make those files readable, avoiding the need to git clone the Pure repository, you can just copy and paste their contents into https://unminify.com and copy that to a file to save. There are numerous other minify/unminifiers out there just a search engine query away.
